📝 Welcome to Flask Todo API

Use the endpoints below via Postman/curl or visit Visual Task Manager

  • GET /api/tasks → List all tasks
  • GET /api/tasks/<id> → Get task by ID
  • POST /api/tasks → Add new task (JSON: {"title": "task name"})
  • PUT /api/tasks/<id> → Update task
  • DELETE /api/tasks/<id> → Delete task
